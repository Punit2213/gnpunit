MySQL MIN() Function

The MIN() function returns the smallest value of the selected column.

SYNTAX

SELECT MIN(column_name)
FROM table_name
WHERE condition;

EXAMPLE

mysql> SELECT MIN(income) AS Minimum_Income FROM employees;

OUTPUT

MySQL MAX() Function

The MAX() function returns the largest value of the selected column.

SYNTAX

SELECT MAX(column_name)
FROM table_name
WHERE condition;

EXAMPLE

mysql> SELECT MAX(income) AS "Maximum Income" FROM employees;

OUTPUT

MySQL COUNT()

The COUNT() function returns the number of rows that matches a specified criterion.

SYNTAX

SELECT COUNT(column_name)
FROM table_name
WHERE condition;

EXAMPLE

mysql> SELECT COUNT(emp_name) FROM employees;

OUTPUT

AVG()

The AVG() function returns the average value of a numeric column.

SYNTAX

SELECT AVG(column_name)
FROM table_name
WHERE condition;

EXAMPLE

mysql> SELECT AVG(working_hours) Avg_working_hours FROM employees;

OUTPUT

SUM()

The SUM() function returns the total sum of a numeric column.

SYNTAX

SELECT SUM(column_name)
FROM table_name
WHERE condition;

EXAMPLE

mysql> SELECT SUM(working_hours) AS "Total working hours" FROM employees WHERE working_hours>=12;

OUTPUT
